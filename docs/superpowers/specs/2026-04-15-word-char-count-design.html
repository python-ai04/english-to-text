Design: Word & Character Count Feature

Дата: 2026-04-15  |  Проект: english-to-text  |  Статус: Approved

Overview

Добавить подсчёт слов и символов (с пробелами и без) прямо под распознанным текстом в интерфейсе Gradio.

Changes

Функция transcribe_image

Возвращает кортеж из 4 значений вместо одной строки:

  1. Распознанный текст (str)
  2. Количество слов (int) — len(text.split())
  3. Символов с пробелами (int) — len(text)
  4. Символов без пробелов (int) — len(text.replace(" ", ""))

Edge case: если изображение None — возвращаем предупреждение и нули.

Gradio Interface

Заменяем один outputs=gr.Textbox(...) на список из 4 компонентов:

outputs=[
    gr.Textbox(label="Распознанный текст", lines=10),
    gr.Textbox(label="Количество слов"),
    gr.Textbox(label="Символов (с пробелами)"),
    gr.Textbox(label="Символов (без пробелов)"),
]

Scope

  • Изменяется только main.py — новых файлов нет.
  • Внешние библиотеки не нужны — только стандартные методы строк Python.
